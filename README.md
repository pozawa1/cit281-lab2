# Part 2
Create a new file, lab-02-node.js. 
```
function square(num) {
    return num*num;
}
console.log('Square operations:')
for (let i = 2; i <= 10; i+=2) {
    console.log(`Square of ${i} is ${square(i)}`);
}

console.log('Square of 20 is ${square(10)}');
```
